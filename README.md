# 🚀 IMPHEN.team — Capstone Project 2026

Selamat datang di repository resmi **IMPHEN.team** untuk Capstone Project 2026.
Repository ini menjadi pusat kolaborasi pengembangan sistem berbasis **Fullstack** serta **Data Science & AI**.

---

## 📌 Deskripsi Proyek

Proyek ini bertujuan untuk membangun solusi digital yang mengintegrasikan:

* 🌐 **Frontend & Backend Development (Fullstack)**
* 🤖 **Data Processing & Artificial Intelligence**

Setiap tim memiliki fokus kerja yang terstruktur untuk memastikan efisiensi dan skalabilitas pengembangan.

---

## 🧩 Struktur Tim & Tanggung Jawab

### 💻 Fullstack Team

📁 Direktori kerja: `/01_DEV-FULLSTACK`

**Tanggung jawab:**

* Pengembangan **User Interface (UI)**
* Integrasi dan konsumsi **API**
* Pengelolaan **Frontend & Backend**
* Menjamin pengalaman pengguna yang optimal

---

### 📊 Data Science & AI Team

📁 Direktori kerja: `/02_LAB-DATA-AI`

**Tanggung jawab:**

* Pengolahan dan analisis data
* Pengembangan model **Machine Learning / AI**
* Eksperimen dan validasi model
* Deployment model ke sistem utama

## ⚙️ Workflow Pengembangan

1. Clone repository
2. Buat branch baru untuk fitur:

   ```
   feature/nama-fitur
   ```
3. Commit dengan pesan yang jelas
4. Ajukan Pull Request (PR)
5. Review oleh tim terkait sebelum merge
